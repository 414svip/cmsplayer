播放器使用说明介绍：
CMS Player是一款安卓应用工具, 简称播放器,
中文名称：席梦思播放器.

版本：CMS Player版本是一款需要自己架设服务器提供接口使用，是一款带有自主播放器后台管理设置， 可设置,广告,弹窗,推广API,以及提供苹果cms影视接口，api订阅服务等自我主管开发产品。

版本：Player DIY版本是一款自己寻找MacCMS俗称苹果cms影视建站系统资源接口的个人播放器， 主打就是自己寻找API源来使用，无需架设服务器等操作，适合有技术懂网络知识的人使用。

版本：Player TV版本是安卓电视机个人播放器，也是需要寻找苹果cms8或10的API接口，sjon数据才能使用。
